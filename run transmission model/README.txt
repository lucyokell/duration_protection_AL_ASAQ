To run the model:
Download all the files in this folder, and store within a single folder.
Right click on the batch files 'aq_lum_runs_updated2.bat' and click Edit.
Within this batch file, after 'set root=' type the file path for the folder where you have stored all these files. 
Save the batch file and shut.
Double click on the batch file in windows explorer to run the model - the output files should appear in the output folder.
